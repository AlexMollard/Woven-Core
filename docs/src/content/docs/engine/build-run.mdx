## Prerequisites

### Vulkan SDK (Latest)

**Why:** This project uses Vulkan 1.4 and extensions that require recent SDKs.

**Required extensions:**
- VK_EXT_shader_object
- VK_EXT_mesh_shader (required for current demo)
- VK_EXT_dynamic_rendering (core in 1.3)
- VK_KHR_synchronization2 (core in 1.3)

**Check compatibility:** [Vulkan GPU Info](https://vulkan.gpuinfo.org/) - search for your GPU and verify extension support.

### Compiler

**Why C++23:** Uses modern features like `std::expected`, ranges, etc.

- **Windows:** MSVC 2022 v17.8+ or Clang-CL
- **Linux:** GCC 13+ or Clang 16+

### CMake 3.28+

**Why recent version:** Uses modern CMake presets and CPM package manager features.

## Build Steps

See [README](README.md) for detailed build commands.

### Quick Start

```bash
# Windows
cmake --preset windows-vs2022
cmake --build --preset windows-debug

# Linux  
cmake --preset linux-debug
cmake --build --preset linux-debug
```

### Presets

Defined in [CMakePresets.json](CMakePresets.json):

- **windows-vs2022**: Generate Visual Studio solution
- **windows-debug/release**: Build configurations
- **linux-debug/release**: Ninja builds

**Why presets?** Standardizes configuration across machines. No manually setting cache variables.

## Running

The executable is in the build folder:
- **Windows Debug:** `build/windows-vs/Debug/WovenCore.exe`
- **Windows Release:** `build/windows-vs/Release/WovenCore.exe`
- **Linux:** `build/linux-debug/WovenCore` or `build/linux-release/WovenCore`

**With Tracy profiling:**
1. Download Tracy server from [GitHub](https://github.com/wolfpld/tracy)
2. Run Tracy server
3. Run WovenCore - it will connect automatically
4. See CPU and GPU timeline

## Troubleshooting

### Validation errors on startup

**Problem:** Validation layers complain about missing features or extensions.

**Fix:** Check that your GPU supports the required extensions. See [Vulkan GPU Info](https://vulkan.gpuinfo.org/).

### Mesh shader not supported

**Problem:** App fails with "Mesh shaders not supported on this device".

**Fix:** Update GPU drivers or use a newer GPU. Mesh shaders require recent NVIDIA/AMD hardware.

### Slang compilation fails

**Problem:** Shader compilation errors on startup.

**Fix:** Check that [shaders/triangle.slang](shaders/triangle.slang) exists and is in the working directory. CMake should copy shaders to the build folder.

## Build Configuration

### Debug vs Release

**Debug:**
- Full validation layers (slow but catches bugs)
- Tracy zones active
- Debug symbols
- No optimizations

**Release:**
- Minimal validation
- Tracy still works but lower overhead
- Full optimizations

**When to use which:** Always use Debug while learning. Use Release to measure real performance.

### CPM Packages

CMake + CPM fetches dependencies automatically:
- SDL3
- Vulkan headers
- vk-bootstrap
- VMA
- Tracy
- Slang
- Jolt Physics
- enkiTS
- GLM
- fastgltf

**No git submodules.** Everything is downloaded on first configure.
