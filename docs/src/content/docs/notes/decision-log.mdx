## How to use this

When you make a significant decision, write:
- **Decision:** What did you choose?
- **Context:** What problem were you solving?
- **Alternatives:** What else did you consider?
- **Trade-offs:** What did you sacrifice? What did you gain?
- **Future:** How might this change later?
